# Analog Clock

A simple analog clock implemented in Java using Swing.

## Features
- Aspect Ratio: 4:3
- Colors: 2 (black and white)
- Shows hour and minute hands
- 12 hour marks

## How to Run
1. Compile the Java files:
   ```sh
   javac Clock.java ClockApp.java
   ```
2. Run the application:
   ```sh
   java ClockApp
   ```

## Files
- `Clock.java`: The clock drawing panel
- `ClockApp.java`: The main application launcher
